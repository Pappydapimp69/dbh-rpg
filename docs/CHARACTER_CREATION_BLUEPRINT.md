# Character Creation Blueprint

This blueprint expands DBH RPG character creation with original playable species inspired by Dragon Ball game design patterns, not copied franchise races or lore.

## Reference Takeaways

- Dragon Ball Xenoverse 2 frames avatar creation around custom appearance, abilities, moveset, and style fit.
- Xenoverse 2 also uses race and mentor identity to create long-term build consequences.
- Dragon Ball Z: Kakarot reinforces that RPG identity comes from fighting, fishing, eating, training, exploration, and bonds.

## Five Stages

1. Race/species model.
2. Stats and XP growth.
3. Special abilities and passives.
4. Creator UI and save integration.
5. Documentation and validation.

## Original Species

- Terran: balanced martial artist with better training and item use.
- Starforged: high-risk battle prodigy with faster boss growth.
- Verdant: regenerative guardian with hidden-technique perception.
- Bio-Arc: precision evolver with efficient blasts and flight-route awareness.
- Majinite: chaotic resilient trickster with loot and status advantages.
- Android: steady reactor fighter with spirit recovery and weather resistance.
- Celestial: late-blooming spirit adept with form-control advantages.

Each species defines starting stat modifiers, per-level growth, XP rate, passives, an active ability, form affinity, starting items, and aura bias.
