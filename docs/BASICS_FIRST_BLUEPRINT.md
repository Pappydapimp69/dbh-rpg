# Basics First Blueprint

This pass pulls the project back toward the fundamentals before adding more advanced RPG layers.

## Focus

The game should be easy to start, understand, move through, interact with, fight in, save, and restart.

## Five Stages

1. Launch flow: start screen, continue, new game, and controls.
2. Save basics: new game actually resets state; continue loads the existing save.
3. Interaction basics: HUD shows what is nearby before pressing interact.
4. Combat basics: attacks report when no target is in range and blasts do not waste spirit into nothing.
5. Validation: smoke checks protect the start flow, reset path, interaction hint, and basic combat feedback.

## Why Now

Character species and transformation mastery add depth, but depth is only useful when the core play loop is clear. This blueprint stabilizes the first minute of play and the repeated actions the player uses constantly.
